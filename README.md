# java-bridge-test

## 기능
- 다리 만드는 기능
  - 위에 다리
  - 밑에 다리
- 다리의 길이 입력 및 저장
- 0 과 1 랜덤 숫자 생성
  - 0 은 아래 칸
  - 1 은 위에 칸
- 다리의 이동할 칸 입력 (다리를 건널 때 U, D 로 정함)
  - 입력한 칸이 랜덤 숫자와 같으면 O 표시
  - 입력한 칸이 랜덤 숫자와 같지 않으면 X 표시
- 입력한 칸과 비교하는 기능
- R or Q 를 입력해서 재시작 여부 확인
  - 재시작 해도 처음 만든 다리로 시작
- 시도한 횟수 더하고 저장하기
- 결과를 출력할 때는 항상 괄호 닫기
  - 다음 결과 입력할 때 괄호 짝대기로 바꾸고, 괄호 닫기
  - 마지막은 항상 괄호 닫기
- 성공 실패 출력
